### Installation
1. Create a virtual environment for the project:
```
python -m venv venv
```
2. Activate the virtual environment:
  - On Windows:
```
venv\Scripts\activate
```

  - On macOS and Linux:
```
source venv/bin/activate
```
3. Install the required dependencies:
```
pip install -r requirements.txt
```

4. run:
```
python main.py
```
